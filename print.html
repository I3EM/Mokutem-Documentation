Mokutem Client API

Introduction

(In Case of Testing) Launching the Mock Tem Server

Launch the mock-temserver.exe executable by double clicking. The mock temserver will then loop, listening for incoming connections.

Launching the Mokutem Server

You may run the mokutem.exe executable by double-clicking. Mokutem will search for a file named mokutem.toml located in its folder. It contains the IP adress of the microscope.

If the file doesn't exist, Mokutem creates a default configuration file with the local ip 127.0.0.1. You can then specify the true IP address if needed.

If Mokutem is stuck, don't hesitate to kill the app and relaunch it.

Connecting with the Mokutem Pling Client

Connecting to the Mokutem Server is as simple as running the following line:

TemConnect()

No argument is needed.
